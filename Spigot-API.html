Spigot-API

Allgemein:

Die Spigot API ist die Grundlage für das Programmieren in Minecraft. Die Vorraussetzung dabei ist die Programmiersprache Java.



Einfacher Command:

Diese Methode wird ausgeführt, wenn ein Spieler einen bestimmten Befehl im Spiel eingibt.

public class [COMMANDNAME] implements CommandExecutor {

public boolean onCommand(CommandSender sender, Command command, String label, String[] args) {

return false;

}
}

Commandsender: Ist der Sender, des Befehls. Das kann ein Spieler, aber auch die Konsole sein.

Commad: Das ist der Befehl selbst.

String: Ist das erste Wort des Befehls.

String[]: sind die Wörter nach dem ersten Wort.
